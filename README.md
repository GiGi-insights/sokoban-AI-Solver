# Sokoban Puzzle: AI Auto-Solver

This repository contains my university project focusing on Artificial Intelligence. It is a classic Sokoban game integrated with an AI agent capable of finding the solution automatically.

### 1. Technical Details
- **Programming Language:** Python 3.x
- **Algorithm Approaches:** Breadth-First Search (BFS), A* Search (A*), and Hill Climbing
- - **Graphics Library:** Pygame

### 2. Game Modes
The project supports two distinct gameplay experiences:
- **Player Mode (Manual):** You can play the game manually by using the `Arrow Keys` (Up, Down, Left, Right) to move the character and push the crates to their targets.
- **AI Auto-Solve Mode:** Press `[Space / Enter]` to activate the AI agent. The Hill Climbing algorithm will instantly calculate the optimal path and automatically execute the moves to solve the puzzle.

### 3. Requirements & Setup
To run this project on your local machine, make sure you have installed the necessary package using pip:

`pip install pygame`

### 4. Execution
Navigate to the project folder via terminal/command prompt and start the main script:

`Sokoban.py`

*(Troubleshooting: If the game doesn't launch properly, please verify that your Python version is 3.8 or newer).*
